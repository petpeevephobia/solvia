
### 2. Add to Dashboard Repository

### 3. Add to Landing Page Repository
- Go to: https://github.com/solviasg/solvia-site/settings/secrets/actions/new
- Name: SERVER_SSH_KEY
- Secret: Paste the same content of id_rsa_solvia
- Click "Add secret"
